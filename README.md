# Redis desktop

---

> 打包前需要生成 [redis-web-module](https://github.com/epldlnwynn/redis-web-module) UI模块

## Getting Started


### NPM 使用

- Install dependencies
```bash
$ npm install
```

- 运行接口服务
```bash
$ npm run server
```


- dist 打包编译桌面应用
```bash
$ npm run build
```
